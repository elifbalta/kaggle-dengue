# Dengue Infection Classification

This project was developed for the STAT412 Kaggle-style competition.

## Objective
Predict dengue infection status using demographic and environmental features.

## Methods
- Feature engineering
- Random Forest
- XGBoost
- Model validation and AUC evaluation

## Tools
R, tidyverse, caret, randomForest, xgboost, pROC

## Data
The dataset is not included due to competition and privacy constraints.
